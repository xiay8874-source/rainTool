② 右键组内标签 → 单页操作菜单

新增/复制/收藏 + 移动到其他分组(二级菜单) + 移出当前组 + 移至新窗口 + 固定 + 关闭。影响单个标签。

两套菜单对应两个对象:
① 组标题菜单(管容器): 改名 / 选颜色 / 组内加标签 / 移组到新窗口 / 全部收藏 / 解散组(保留标签) / 关闭组 / 删除组
② 组内标签菜单(管个体): 下方新增 / 复制 / 收藏 / 移动到其他分组 ▸ / 移出当前组 / 移至新窗口 / 固定 / 关闭
关键差异: ① 改色改名解散是组级操作;② 移出/移动到其他组是个体在组环境下的位置调整
