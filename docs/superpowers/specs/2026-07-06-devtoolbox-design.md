# DevToolbox 设计文档

> 程序员开发工具百宝箱 — Electron 桌面应用

## 1. 产品定位

- **形态**:Electron 桌面应用(macOS / Windows / Linux)
- **用户**:个人自用,后端开发(Java 为主)
- **目标**:把日常开发反复查的小工具做成一个本地桌面应用,隐私好、可离线、可访问本地文件

## 2. 技术栈

| 层 | 选型 | 理由 |
|---|---|---|
| 运行时 | Electron | JS 一把梭,前端生态全用,个人自用不担心体积 |
| 框架 | React 18 + TypeScript | 类型安全,生态成熟 |
| 状态 | Zustand | 轻量,适合标签/分组/收藏的复杂嵌套状态 |
| 样式 | Tailwind CSS | 快速落地设计系统,原子类便于调对比度 |
| 构建 | Vite + electron-builder | 快速 HMR,打包多平台 |
| 持久化 | electron-store(收藏/配置)+ localStorage(标签会话) | 收藏需跨重启持久;标签会话可临时 |

## 3. 视觉设计语言(简约淡雅 / 低对比度)

这是核心约束,所有界面遵循。

### 配色(低饱和、低对比度)

| Token | 值 | 用途 |
|---|---|---|
| `--bg-app` | `#fafafa` | 应用底色 |
| `--bg-surface` | `#ffffff` | 卡片/工作区表面 |
| `--bg-subtle` | `#f4f5f7` | 侧栏底、悬浮区 |
| `--bg-hover` | `#eef0f3` | hover 态 |
| `--border` | `#ebecef` | 细分割线 |
| `--border-strong` | `#e0e2e6` | 输入框边等稍重 |
| `--text-primary` | `#3a3d42` | 主文字(非纯黑) |
| `--text-secondary` | `#6b7178` | 次要文字 |
| `--text-tertiary` | `#9aa0a8` | 占位、提示 |
| `--accent` | `#6b7c93` | 强调色(低饱和灰蓝,非高饱和靛蓝) |
| `--accent-bg` | `#eef1f5` | 强调背景(选中态) |
| `--danger` | `#b08584` | 删除(低饱和暗红) |

**原则**:无纯黑、无纯白边、无高饱和色;强调仅用于"当前选中"等极少数场景;分组颜色一律用低饱和莫兰迪色。

### 字号(克制)

| 用途 | 大小 | 字重 |
|---|---|---|
| 页面标题 | 13px | 500 |
| 正文/标签 | 12px | 400 |
| 次要说明 | 11px | 400 |
| 标签组名/小标 | 10px | 500 |
| 等宽代码 | 12px | 400 |

无 16px 以上大字;无粗体黑字(700)。

### 其他

- 圆角 6px(卡片)、4px(按钮/输入)
- 阴影仅悬浮窗使用:`0 8px 24px rgba(0,0,0,.06)`,普通元素无阴影
- 间距以 4 的倍数:4 / 8 / 12 / 16 / 24
- 分组颜色板(低饱和):`#a8b0bd` `#a3a89c` `#9aa6b0` `#b0a3a1` `#a5a8b5` `#9eada6`

## 4. 工具清单(按优先级)

### P0 — 第一版必做

| 分类 | 工具 | 说明 |
|---|---|---|
| 编码转换 | Base64 / URL / Unicode / HTML 实体 编解码 | 后端最高频 |
| 格式化 | JSON 格式化与压缩、SQL 美化 | 后端日常 |
| 加密哈希 | MD5 / SHA 系列 / HMAC、AES 加解密 | 排错签名 |
| 时间 | 时间戳 ⇄ 日期(多时区) | 日志分析 |
| 代码生成 | JSON → Java POJO | 后端用 Java |

### P1 — 高价值第二批

正则测试、JWT 解析、CSV/JSON/YAML 互转、文本 Diff、SQL 建表 → Java Entity、HTTP 请求测试

### P2 — 锦上添花

UUID、密码生成、.gitignore 生成、cron 解析、颜色转换、图片 Base64/压缩

### P3 — 暂缓

二维码、DNS/IP 查询、端口检测

## 5. JSON 工作台(核心模块)

JSON 是核心中的核心,做厚做深。

### 功能

1. **容错解析** — 即便 JSON 不完全合法(缺引号、尾逗号、单引号、注释)也尽力展开,标记错误位置
2. **树形浏览** — 可折叠/展开,大 JSON 虚拟滚动流畅渲染
3. **Diff 对比** — 两份 JSON 高亮增/删/改差异
4. **实时编辑** — 树视图 + 文本视图双视图,双向同步,实时校验

### 实现

- 容错解析用 `json5` + 自定义修复策略(去尾逗号、单→双引号、剥离注释)
- 树渲染用虚拟列表(`react-virtual`)
- Diff 用 `diff` 库做结构化差异

## 6. 界面布局

三栏结构:

```
┌──────┬─────────────┬──────────────────────────┐
│ 图标 │  标签页侧栏  │        工作区            │
│ 栏   │ (可收起)    │  (工具栏 + 内容)         │
│      │             │                          │
└──────┴─────────────┴──────────────────────────┘
```

- **图标栏**(最左,~48px):工具类型图标,点击选中当前工具类型,在标签侧栏新建该类型标签
- **标签页侧栏**(~220px,可收起为 ~20px):展示已开标签,按分组组织
- **工作区**(自适应):顶部工具栏 + 工具内容

### 收起/呼出

- 标签侧栏可收起为窄条(显示标签数),点 ≫ 呼出
- 收起时工作区最大化

## 7. 标签页与分组系统

参考 Chrome Tab Groups。

### 自动分组与自定义分组

- **自动分组**:新建标签默认按工具类型归组(JSON 工作台、加密哈希、编码转换…)
- **自定义分组**:可新建分组并命名,把不同类型标签拖进去混合存放
- 每个标签独立状态(输入/配置互不影响)
- "复制此页"克隆当前页全部内容到新标签

### 组标题菜单(管容器)

点击组标题弹出:
- 分组名称输入框
- 颜色选择(6 色莫兰迪)
- 在组内添加新标签页
- 将分组移至新窗口
- ★ 全部保存到收藏夹
- 调整分组顺序
- 解散标签组(保留标签,落入未分组)
- 关闭标签组 / 关闭并删除标签组

### 组内标签菜单(管个体)

右键组内标签弹出:
- 在下方新增标签页
- 复制此页
- 收藏此页
- 移动到分组 ▸(二级菜单:列出其他组 + 新建分组)
- 移出当前标签组
- 将标签页移至新窗口
- 固定标签页
- 关闭标签页

### 标签改变分组 — 三种方式

1. **拖拽** — 拖到目标分组,目标高亮提示,松手移入
2. **右键** → 移动到分组 ▸
3. **工具栏** "移动到分组"下拉

### 重命名

- **标签页**:双击标签名或右键 → 重命名
- **分组**:组标题菜单内输入框直接改

### 未分组区

兜底机制,标签移出分组或分组被删后自动落入,可再分配。

## 8. 收藏夹(悬浮窗口)

### 形态

- **悬浮窗**:浮于工作区之上,不占布局
- **拖拽**:拖标题栏可在应用内任意移动,记住位置
- **尺寸**:可拖边角调整大小;含搜索框 + 滚动列表
- **唤起/收起**:左侧 ★ 图标点击,或 ⌘B 快捷键;关闭(✕)后图标态变化提示
- **失焦行为**:点窗外区域不自动关闭(可配置"失焦自动收起")
- 阴影:`0 8px 24px rgba(0,0,0,.06)`

### 收藏粒度

- **单个标签**:保存工具类型 + 输入内容
- **整个分组**:保存分组名 + 组内所有标签 + 归属结构

两个入口(工作区工具栏):
- "★ 收藏此页" → 存单页
- "★ 收藏当前分组" → 存整组

### 收藏夹内分组

- 与标签页分组是两套独立体系
- 可建收藏分组整理(如"常用 JSON 模板""线上环境调试")

### 完整恢复

- 恢复分组收藏时,原样重建:分组名、组内所有标签、各标签工具类型与内容、归属关系
- 支持两种方式:追加到当前标签栏 / 替换当前

### 持久化

存本地(electron-store),重启不丢。

## 9. 项目结构

```

├── electron/                  # 主进程
│   ├── main.ts                # 窗口创建、应用生命周期
│   └── preload.ts             # IPC 桥(electron-store 读写)
├── src/                       # 渲染进程(React)
│   ├── components/
│   │   ├── layout/            # 三栏布局、图标栏
│   │   ├── tabs/              # 标签侧栏、分组、两套右键菜单
│   │   ├── favorites/         # 悬浮收藏夹
│   │   └── tools/             # 各工具
│   │       ├── json-workbench/
│   │       ├── crypto/
│   │       ├── encode/
│   │       └── ...
│   ├── store/                 # Zustand: tabs, groups, favorites
│   ├── lib/                   # 工具函数(JSON容错解析、加解密等)
│   └── styles/                # Tailwind config + tokens
├── docs/superpowers/specs/    # 本设计文档
├── package.json
└── vite.config.ts
```

## 10. 实施顺序

1. **脚手架**:Electron + React + TS + Vite + Tailwind,跑通 hello world
2. **设计系统**:低对比度 tokens、基础组件(卡片/按钮/输入)
3. **三栏布局 + 图标栏**:静态结构
4. **标签页系统**:侧栏、多标签、自动/自定义分组、收起/呼出
5. **分组交互**:拖拽改组、两套右键菜单、重命名、颜色
6. **收藏夹悬浮窗**:收藏/恢复、分组收藏、完整恢复
7. **P0 工具**:编码、格式化、加密、时间、JSON→POJO
8. **JSON 工作台**:容错解析、树形、Diff、编辑
9. **P1/P2 工具**:逐步补齐
